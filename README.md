# Transparent Siri

A conceptual Apple Intelligence / Siri interaction prototype built using pure HTML, CSS, and JavaScript.

This prototype explores a more transparent AI assistant experience where Siri clearly explains:
- what it understood
- what actions it plans to take
- and asks for approval before executing tasks

The goal was to make AI interactions feel more trustworthy, understandable, and controllable.

---

## Features

- Dynamic Siri-style UI
- Simulated Apple Intelligence parsing
- Expandable “Here’s what I understood” section
- Action approval workflow
- Context-aware request parsing
- Animated typing indicators
- Multiple request types supported:
  - Restaurant reservations
  - Calendar events
  - Reminders
  - Messaging
  - Navigation
  - Music playback
  - Smart home controls
  - Playlist generation
  - Information lookups

---

## Built With

- HTML
- CSS
- Vanilla JavaScript

No frameworks or external libraries were used.

---

## Design Goals

This prototype was designed to explore:
- AI transparency
- Human-AI trust
- Approval-based automation
- Conversational UX
- Apple-inspired interaction patterns

Instead of silently executing actions, the assistant communicates intent before taking action.
